### 如何運行專案

#### clone 專案

1. clone 專案

```bash
git clone https://github.com/AndreaFan123/homework.git
cd homework
```

2. 使用 node:20.12.2 開發環境

```bash
nvm use 20.12.2
```

3. 啟動專案

```bash
npm run dev
```
